# Console Banking System (Java)

A console-based banking system built using Core Java to demonstrate Object-Oriented Programming (OOP) concepts like classes, objects, and modular design.

## Features
- Create Account
- Deposit Money
- Withdraw Money
- Basic Transaction Handling
- Transaction History (In Progress)

## Tech Stack
- Java (Core Java)
- OOP Concepts

## Project Structure
- model → Contains Account & Transaction classes
- service → Business logic (BankingService)
- Main.java → Entry point

##  Project Status
In Progress – actively adding new features

## Learning Goals
- Understand OOP concepts deeply
- Practice real-world project structure
- Build a strong GitHub portfolio project
